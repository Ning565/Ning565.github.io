2026全年规划

发表于 2026-03-07 更新于 2026-04-17 分类于 研究生生涯

2026规划

直到写下这个规划的这一刻，我才意识到2026已经过去了1/4了，时间过的好快，但可以预见的是，这个2026越往后会过的越快。

我是一个很喜欢做规划的人，但不管是由于自身的原因，亦或是各类变故，一段时间后我经常落实不了自己的规划，但总归做规划可以让我安心，让我为自己打一下鸡血，哪怕一刻也是有价值的。

整个2026，有如下三个方面的目标和规划：

找工作

首先，2026年最大的关键词就是找工作，2026轮到我秋招了，前面努力那么久，经历的那么多痛苦，都是为了最终的找工作目的。

从27届开始的明显趋势就是，暑期实习成为秋招的主战场了，大部分的hc都是靠转正实习来释放出去，而不是正式的秋招流程了，越好的部门，越好的工作（大模型算法人才计划除外）这种现象越明显，所以找好一个暑期实习是至关重要的。

关于实习的规划，现在整体的趋势都是大力发展AI，大模型泡沫还在加杠，并且看起来短时间内（3年内）不太会落下，资本前期投入的足够多，已经没法下来了，必须一直投，就像传销式的；但对于公司来说，主要的盈利（优化）点在于可以优化掉相当一部分人力，而不是实现to C的盈利。架不住资本一直在砸钱，而且3年以后的事情谁能说得准呢，因此现在看来，选择大模型相关的工作是非常必须的。

因此，我的暑期实习的最终目标就很确定了：大模型相关业务 + 转正率高 + base北京

但投递的时候确实可以多投一些其他base的地方，练一下面试总归是好事；

正式工作：因为现在的规划是不长留北京，但是找工作又必须找北京的，因此目前的想法是先在北京待上个1-2年，最多两年就要逃离北京，跳到杭州的互联网公司（目前的最优选）。所以找正式工作的时候，业务大抵还得是大模型相关的，并且对新人来说有着不错的发展前景，有新的业务可以给到新人去做，能短时间内有明确产出的，并且薪资待遇需要是不错的，方便后续跳槽涨薪，认清自己的定位，大厂赚的就是这几年的辛苦费

但是我深知自己没有任何大模型相关的经历，没论文，没实习，没相关项目，其实找到大模型相关的岗位还是比较困难的，同时之前的力扣基础、八股啥的都忘得差不多了，因此给自己的近期规划就是：

  • 按准备好的简历来准备面试需要的知识，技术栈的八股知识点，两段实习的技术细节深挖，过一遍第二段实习的每一个项目点，整体问题和思考

  • 做一个大模型agent的项目，学习hello-agents，做一个项目

  • 每日力扣，可以不多，但得坚持，好像有点来不及了

  • 学习大模型的知识，注意复盘

锻炼

在2026年2月8日，GALA发微博说：”健身是为数不多努力就会有收获的事“，我希望这一年通过自己的努力可以有所收获，让我真正坚持下来一件事。

整年的核心目标主要有两块：

减脂：回到125斤作用并保持住

主要动作/内容：做有氧，跑步爬坡（主要在控制心率），控制饮食（不暴饮暴食）

增肌：增加肌肉，作为爱好

主要动作：

  • 胸：卧推 + 哑铃上斜卧推 + 蝴蝶机夹胸 + 坐姿器械推胸（如果有）
  • 背：辅助引体 +跪姿哑铃划船 + 高位下拉
  • 腿：杠铃深蹲 + 器械内外收
  • 肩：哑铃坐姿推肩/ 史密斯推肩（前束） + 站姿飞鸟 （中束） + 蝴蝶机反向飞鸟（后束）
  • 核心：卷腹（仰卧起坐） + 蝴蝶屈腿 + 剪刀提髋 + 俄罗斯转体 + 蹬自行车 + 钟摆 + V字举

科研/毕业

第一篇小论文的实验，迅速做完几个实验，写完小论文，投出去

在找到实习以后，立刻开始准备中期答辩的内容和实习实践的内容，该写的几个内容都写完

找完工作以后就要着手大论文的准备了，不能那么随意，拖拉

总结

纵向维度，每日的计划：要保证早睡早起（控制自己不牺牲睡眠），保证每天8.00-8.30起床，雷打不动的规矩

  • 早起吃饭，健身，大概10点左右开始工作，保持早睡早起
  • 每日7h的科研工作
  • 每日3h的找工作学习

健身：三分化

一天胸+三头、一天背+二头、休息一天（练下肩）

一天胸+三头、一天背+二头、一天有氧+肩、休息一天

我已经尽可能做到了戒骄戒躁，谦虚谨慎；现在还需要做的是思考后再说话，思考后再行动，不要盲目，乱撞，要带着思考和目的去行动
