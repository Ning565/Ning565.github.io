Markdown笔记

标题

标题用两个“## XX”加回车即出来,几个#表示几级标题，勿忘空格！

字体

加粗 ：两个 倾斜* ：一个 三个就是加粗加倾斜啦 删除线：两个~

引用

搭建自己的博客，记录自己的笔记，走自己的路

用一个英文的>来表示引用

法1：手动分割(bushi两个~~前后

公式

\(a = b \times c\)

\(L_p = (\sum_{i=1}^m |x_i - y_i |^{p} )^{ {\tfrac{1} {p} } }\)

图片
